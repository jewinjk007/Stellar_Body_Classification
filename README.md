## Stellar Body Classification

A machine learning project for classifying celestial bodies using astronomical data and classification algorithms.

### Overview

This project implements a classification system to identify and categorize different types of stellar bodies (stars, galaxies, quasars, etc.) using machine learning techniques. The analysis is conducted in a comprehensive Jupyter Notebook that includes data exploration, feature engineering, model training, and evaluation.

### Features

- **Data Analysis**: Exploratory data analysis of astronomical datasets
- **Feature Engineering**: Extraction and preprocessing of stellar characteristics
- **Machine Learning Models**: Multiple classification algorithms and comparisons
- **Visualization**: Detailed visualizations of results and model performance
- **Evaluation Metrics**: Comprehensive performance analysis and validation

### Requirements

- Python 3.7+
- Jupyter Notebook
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/jewinjk007/Stellar_Body_Classification.git
cd Stellar_Body_Classification
```

2. Install required packages:
```bash
pip install jupyter numpy pandas scikit-learn matplotlib seaborn
```

### Usage

1. Launch Jupyter Notebook:
```bash
jupyter notebook
```

2. Open `Celestial_Body_Classification_Project.ipynb`

3. Run the cells sequentially to:
   - Load and explore the astronomical data
   - Preprocess and engineer features
   - Train and evaluate classification models
   - Visualize results and performance metrics

### Results

The notebook includes detailed analysis of model performance, classification accuracy, confusion matrices, and feature importance analysis.
